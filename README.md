
# CryptoVision AI - Final Submission

## Overview
CryptoVision AI is a professional cryptocurrency forecasting and risk analytics system.
It integrates time-series forecasting models and financial risk metrics into an interactive dashboard.

## Features
- Multi-crypto support (BTC, ETH, SOL)
- ARIMA Forecasting
- Facebook Prophet Forecasting
- Annualized Volatility
- Value at Risk (VaR 95%)
- Monte Carlo Price Simulation
- Interactive Streamlit Dashboard

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run application:
   streamlit run app.py

## Technologies Used
Python, Streamlit, Plotly, ARIMA, Prophet, NumPy, yFinance
